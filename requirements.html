Minimum requirements for MCEdit-Unified:

Windows Mac Linux
Operating System XP SP3 or newer OSX 10.6.8 (Snow Leopard)* or newer Any variant with python 2.7
Processor Intel: Core 2 Duo or better**
AMD: K10 or better 		Intel: Core 2 Duo or better** Intel: Core 2 Duo or better**
AMD: K10 or better
Ram 2GB or higher
Video Card Any OpenGL 3.3 compatible graphics card
Free Disk Space 1 - 1.5GB

Recommended requirements for MCEdit-Unified:

Windows Mac Linux
Operating System 7, 8 and 8.1 (64-bit) OSX 10.9.5 (Mavericks) or newer Any variant with python 2.7
Processor Intel: Core i3 (Sandy Bridge) or better
AMD: Bulldozer or better 		Intel: Core i3 (Sandy Bridge) or better Intel: Core i3 (Sandy Bridge) or better
AMD: Bulldozer or better
Ram 4GB or higher
Video Card Any OpenGL 4.3 compatible graphics card
Free Disk Space 1 - 1.5GB

Notes:
*Not thoroughly tested with 10.9.5 Mavericks or below.
**Due to the speed on how fast MCEdit-Unified can run and any other programs that are currently on without crashing.
If you are currently running MCEdit-Unified on Intel Core 2 Duo or lower you may recieve a crash on newer released versions of MCEdit-Unified.
